# PurrAi CMS Product Overview

PurrAi is an enterprise-grade, multi-tenant headless CMS built for scalable content management with advanced security and internationalization.

## Core Capabilities

- **Multi-tenancy**: Database-per-tenant architecture with automatic domain routing via `stancl/tenancy`
- **Headless CMS**: Full REST API with Sanctum authentication for decoupled frontends (React, Vue, mobile)
- **Content Management**: Dynamic page builder with drag-and-drop blocks, custom components, and collections
- **Internationalization**: Full multilingual support (en_US, pt_PT, es_ES, pt_BR) with locale-aware content
- **Security**: RBAC with Shield, 2FA, password confirmation, activity logging, and API token abilities
- **Admin Panel**: Filament 4-based admin interface with custom resources and workflows

## Key Features

- Dynamic collection schemas for custom content types
- Component-based page builder with reusable layouts
- Media library with Curator integration
- Form builder with public submission handling
- Menu builder for navigation management
- Settings system for tenant-specific configuration
- Real-time broadcasting with Reverb
- Background job processing with queue workers
- Activity logging and monitoring with Pulse

## Target Use Cases

- Multi-tenant SaaS applications requiring isolated data
- Headless CMS for modern frontend frameworks
- Enterprise content platforms with complex permissions
- Multilingual websites and applications
- Custom CMS solutions with dynamic content types
